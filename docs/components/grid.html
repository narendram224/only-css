Grid System

The Only-CSS grid system is a powerful, flexible 12-column grid built with CSS Grid. It provides responsive layouts with mobile-first approach and intuitive naming conventions.

Good to know: The grid system uses CSS Grid for modern browser support while maintaining fallbacks for older browsers. It's designed to be lightweight and intuitive.

Basic Grid

The grid system is based on a 12-column layout. Use .oc-row for the container and .oc-col-{number} for columns:

API Reference

Container Classes

Class Description
oc-row Container for grid columns, creates a flex row
oc-container Fixed-width container for centering content
oc-container-fluid Full-width container that spans the entire viewport

Column Classes

Class Description
oc-col-1 to oc-col-12 Fixed width columns (1-12 out of 12)
oc-col Auto-width column that shares space equally
oc-col-auto Column that takes only the space it needs

Offset Classes

Class Description
oc-offset-1 to oc-offset-11 Offset columns by 1-11 column widths

Responsive Breakpoints

Breakpoint Prefix Min Width Description
Extra Small oc-col-xs- 0px Default (mobile-first)
Small oc-col-sm- 576px Small screens (tablets)
Medium oc-col-md- 768px Medium screens (landscape tablets)
Large oc-col-lg- 992px Large screens (desktops)
Extra Large oc-col-xl- 1200px Extra large screens

Best Practices

Grid Best Practices:
  • Always use rows inside containers
  • Keep column numbers consistent across rows
  • Use responsive classes for mobile-first design
  • Avoid nesting grids too deeply (max 2-3 levels)
  • Use semantic HTML5 elements with grid classes

Do's and Don'ts

✅ Do

  • Use semantic HTML with grid classes
  • Plan responsive breakpoints
  • Keep column sums to 12 or less
  • Test on all screen sizes
  • Use containers for proper spacing

❌ Don't

  • Nest grids more than 3 levels deep
  • Mix fixed and auto columns unnecessarily
  • Use grid for simple single-column layouts
  • Override grid widths with custom CSS
  • Forget to test on mobile devices

Customization

The grid system can be customized using CSS variables:

:root { /* Grid breakpoints */ --oc-grid-breakpoint-sm: 576px; --oc-grid-breakpoint-md: 768px; --oc-grid-breakpoint-lg: 992px; --oc-grid-breakpoint-xl: 1200px; /* Grid gutters */ --oc-grid-gutter: 1rem; --oc-grid-gutter-sm: 0.5rem; --oc-grid-gutter-lg: 1.5rem; /* Container max widths */ --oc-container-sm: 540px; --oc-container-md: 720px; --oc-container-lg: 960px; --oc-container-xl: 1140px; }
Pro Tip: You can create custom grid utilities by extending the CSS with your own classes and variables for specific project needs.
