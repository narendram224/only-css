Switch

Switches are toggle controls that allow users to turn settings on or off. Only-CSS provides accessible switch components with smooth animations and proper keyboard support.

Good to know: Switches use semantic HTML checkbox inputs with CSS styling, ensuring maximum accessibility and form integration.

Accessibility

Accessibility Features:
  • Semantic HTML checkbox input
  • Keyboard navigation support
  • Screen reader friendly
  • Focus states clearly visible

Best Practices

  • Always provide clear labels for switches
  • Use switches for binary settings (on/off)
  • Consider default states carefully
  • Test with keyboard navigation
  • Use appropriate color contrast

Design Guidelines

Pro Tip: Use switches for settings that take effect immediately. For settings that require confirmation, consider using checkboxes instead.

When to Use Switches

  • Settings: Enable/disable features
  • Preferences: User preferences
  • Notifications: Turn notifications on/off
  • Features: Enable/disable functionality
