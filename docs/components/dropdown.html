Dropdown

Dropdowns are toggleable menus that allow users to select from a list of options. Only-CSS provides dropdown components using pure CSS with :focus-within for accessibility.

Good to know: Dropdowns work without JavaScript using CSS :focus-within pseudo-class, ensuring keyboard accessibility and screen reader support.

Basic Usage

The simplest way to create a dropdown is to use the oc-dropdown class with proper structure:

Accessibility

Accessibility Features:
  • CSS :focus-within for keyboard navigation
  • Semantic HTML structure
  • Screen reader friendly
  • Focus management handled by CSS
  • High contrast colors

Best Practices

  • Use clear, descriptive trigger text
  • Group related items logically
  • Use dividers to separate item groups
  • Test with keyboard navigation
  • Ensure sufficient color contrast

Design Guidelines

Pro Tip: Keep dropdown menus concise (5-7 items maximum) for better usability. Use sub-menus for complex hierarchies.

When to Use Dropdowns

  • Navigation: Site navigation menus
  • Actions: Contextual actions for items
  • Forms: Select from predefined options
  • Settings: Configuration choices
  • Filters: Content filtering options

Content Guidelines

  • Keep item text short and clear
  • Use consistent ordering
  • Group related items together
  • Use icons for quick recognition
  • Consider mobile touch targets
