Tabs

Tabs are a navigation pattern that allows users to switch between different content panels within the same context. Only-CSS provides a pure CSS tab system using modern CSS Grid and the details/summary elements.

Good to know: These tabs work without JavaScript by using CSS Grid and the HTML5 details/summary elements, making them lightweight and accessible.

Best Practices

  • Use clear, concise tab labels
  • Ensure tab content is relevant to the label
  • Consider the default open tab carefully
  • Test with keyboard navigation
  • Use consistent tab ordering

Design Guidelines

Pro Tip: Tabs work best for organizing related content that doesn't need to be visible simultaneously. Keep tab labels short and descriptive.

When to Use Tabs

  • Product Pages: Description, specs, reviews
  • Settings: Different configuration categories
  • Documentation: Related topics or sections
  • Forms: Multi-step processes
