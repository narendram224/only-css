Progress

Progress bars show the completion status of a task or process. Only-CSS provides progress components with customizable colors, sizes, and animated transitions.

Good to know: Progress bars use semantic HTML5 progress elements with proper ARIA attributes for accessibility.

Accessibility

Accessibility Features:
  • Semantic HTML5 progress element
  • Proper ARIA attributes
  • Screen reader announcements
  • High contrast colors

Best Practices

  • Always provide labels for progress bars
  • Use appropriate colors for context
  • Show percentage or fraction when possible
  • Consider indeterminate progress for unknown duration

Design Guidelines

Pro Tip: Use progress bars for operations with predictable duration. For indeterminate operations, consider using spinners instead.

When to Use Progress Bars

  • File Uploads: Show upload completion
  • Form Steps: Multi-step form progress
  • Task Completion: Show task status
  • Data Processing: Show processing status
