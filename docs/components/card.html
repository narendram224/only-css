Card

Cards are flexible containers for grouping related content. Only-CSS provides a clean card component with header, content, and footer sections, plus hover effects for enhanced interactivity.

Good to know: Cards use semantic HTML5 structure with proper sectioning elements for better accessibility and SEO.

API Reference

CSS Classes

Class Description
oc-card Base card component class
hover Adds hover effects (elevation and shadow)
oc-card-header Card header section (optional)
oc-card-content Main card content area
oc-card-footer Card footer section (optional)

Accessibility

Accessibility Features:
  • Semantic HTML5 structure with proper sectioning
  • Keyboard accessible content
  • Screen reader friendly
  • Proper heading hierarchy

Best Practices

  • Use semantic heading elements in card headers
  • Ensure interactive elements are properly labeled
  • Maintain consistent card layouts within grids
  • Use hover effects sparingly for better performance
  • Ensure sufficient color contrast for text
  • Test cards with assistive technologies

Design Guidelines

Pro Tip: Use cards to group related information and create visual hierarchy. Cards work best when they have a clear purpose and consistent structure.

When to Use Cards

  • Content Grouping: Group related information together
  • Dashboards: Display stats and metrics
  • Product Listings: Show products with images and details
  • Blog Posts: Display article previews
  • User Profiles: Show user information and actions
  • Task Management: Display tasks and their status

Layout Guidelines

  • Use consistent spacing within cards
  • Align cards properly in grid layouts
  • Maintain consistent card heights in rows
  • Use responsive design for mobile devices
  • Consider card loading states
  • Test with different content lengths

Content Guidelines

  • Keep card titles concise and descriptive
  • Use clear and scannable content
  • Include relevant images or icons when helpful
  • Provide clear calls-to-action in footers
  • Use consistent typography and spacing
