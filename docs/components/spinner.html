Spinner

Spinners are loading indicators that show users that content is being processed or loaded. Only-CSS provides animated spinner components with multiple sizes and styles.

Good to know: Spinners use pure CSS animations, making them lightweight and performant without any JavaScript dependencies.

Accessibility

Accessibility Features:
  • Use with appropriate ARIA labels
  • Announce loading states to screen readers
  • Provide alternative text when needed

Best Practices

  • Use appropriate spinner sizes for context
  • Combine with loading text for clarity
  • Don't use spinners for quick operations
  • Provide feedback when loading completes

Design Guidelines

Pro Tip: Use spinners for operations taking more than 1 second. For faster operations, consider other loading indicators.

When to Use Spinners

  • Data Loading: Fetching content from servers
  • Form Submission: Processing form data
  • File Uploads: Uploading files
  • Content Generation: Processing content
