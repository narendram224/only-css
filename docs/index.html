Introduction

Only-CSS is a modern, lightweight CSS component library that helps you build beautiful web interfaces faster. With zero JavaScript dependencies and a focus on semantic HTML, it provides everything you need to create responsive, accessible websites.

⚡
Lightweight
Only ~20KB minified CSS with zero JavaScript bloat. Load times are incredibly fast.
🎨
Modern Design
Clean, modern components with consistent design system and smooth animations.
📱
Responsive
Mobile-first approach with responsive grid system and flexible components.
♿
Accessible
Built with semantic HTML5 and ARIA attributes for maximum accessibility.
🌙
Dark Mode
Built-in dark mode support that respects user preferences.
🔧
Customizable
Easy to customize with CSS variables and modular component imports.

Installation

Option 1: CDN (Recommended for quick start)
!-- Include Only-CSS from CDN ; link rel="stylesheet" href="https://unpkg.com/only-css@latest/dist/index.css"
Option 2: NPM/Yarn
# Using npm npm install only-css # Using yarn yarn add only-css # Using bun bun add only-css
Option 3: Download

Download the latest release from GitHub and include the CSS file in your project.

Browser Support

Only-CSS supports all modern browsers:

  • Chrome 60+
  • Firefox 55+
  • Safari 12+
  • Edge 79+

Note: Some components use modern CSS features like CSS Grid, Custom Properties, and :focus-within. Ensure your target browsers support these features.
